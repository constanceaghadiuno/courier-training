# Medical Courier Training - Study Guide

A free, modern study guide for medical courier compliance training. Covers HIPAA, bloodborne pathogens, and core competencies without the corporate nonsense.

## What This Is

This is supplementary educational material for people who need to take medical courier training. It won't replace your employer's required courses, but it will make them less painful by explaining things clearly.

**Disclaimer:** This does not replace employer-required or state-mandated training and does not constitute certification.

## Features

- ✅ 25 chapters on core competencies
- ✅ 31 chapters on HIPAA compliance  
- ✅ 13 chapters on bloodborne pathogens
- ✅ Progress tracking (saved in browser)
- ✅ Mobile-friendly
- ✅ No sign-up required
- ✅ Completely free

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide Icons

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

### Vercel
1. Push to GitHub
2. Connect repository in Vercel
3. Deploy (zero config needed)

### Netlify
1. Push to GitHub
2. Connect repository in Netlify
3. Build command: `npm run build`
4. Publish directory: `dist`

### Lovable/Replit
1. Import this project
2. Run `npm install`
3. Run `npm run dev`
